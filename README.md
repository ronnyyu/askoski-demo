# askoski-demo
A simple demo for askoski backend and frontend.

```
askoski-demo
├── README.md   -- readme
├── backend     -- backend module
│       ├── api         -- NodeJS api directory
│       ├── model       -- Keras (empty)
│       └── service     -- Flask service directory
├── doc         -- doc    
└── frontend    -- Angular frontend module 
```
